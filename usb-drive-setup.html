Samsung USB Drive Setup Guide

Making Your Drive Secure & iPhone Compatible
One-time setup: ~10 minutes

A Direct Solution for Your Samsung USB Drive

The issue you've described with the Samsung USB stick is a very common and understandable roadblock.

Good News: It is almost certainly not a problem with the drive itself. A high-quality, standard USB drive from Samsung is perfectly suitable for this task.

The difficulty arises from a non-intuitive but crucial setting within the Mac's Disk Utility software. By default, the application hides the specific formatting option needed to make a drive both encrypted and compatible with an iPhone or iPad's native Files app.

This guide provides the direct solution to correctly format the drive. Following these instructions will transform the Samsung USB stick into the secure, portable, and easy-to-use video portfolio your mentor requires.

The Step-by-Step Formatting Guide

1
Connect the Drive
Plug the Samsung USB-C flash drive directly into a USB port on the Mac.
2
Open Disk Utility
  • Click on the Finder icon (the smiling face in the Dock)
  • Open the Applications folder
  • Find and open the Utilities folder inside it
  • Double-click on Disk Utility to launch it
3
Reveal All Devices
CRUCIAL STEP: This is the most important step and the one that is most often missed!
In the menu bar at the very top of the screen, click on View and select "Show All Devices" from the dropdown menu. This changes the sidebar to show the physical drive itself, not just the pre-formatted space on it.
4
Select the Top-Level Drive
In the sidebar on the left, under the "External" heading, click to select the top-level entry for the Samsung USB drive. It will have the manufacturer's name and capacity.
Important: Do NOT select the indented volume name below it - select the parent drive!
5
Erase and Format
With the top-level drive selected, click the "Erase" button in the toolbar. Configure as follows:
  • Name: Give it a simple name like Show and Tell
  • Scheme: Select GUID Partition Map
  • Format: Choose APFS (Encrypted)
6
Create Your Password
  • Enter a memorable but secure password
  • Enter it again to verify
  • Click "Choose" then "Erase"
CRITICAL: Write down this password and store it in a secure, private location. If this password is lost, there is no way to recover the data on the drive!
7
Completion
The Mac will reformat the drive with the secure file system. Once finished, click "Done".
The drive is now fully prepared and ready to use!

Using the Secure Drive

Once prepared, using the drive is simple:

On the Mac

Plug the drive in and enter the password when prompted. You can then drag and drop video files onto the drive's icon.

On the iPhone/iPad

Connect the drive (using an official Apple adapter if the device has a Lightning port). Open the blue Files app, tap on the drive's name under "Locations," and enter the password when prompted to view and play the videos.

This process should resolve the issue entirely, making the Samsung USB drive work exactly as intended.
If for any reason this procedure is unsuccessful, we will investigate further.
